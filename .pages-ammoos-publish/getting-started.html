Getting Started

AmmoOS 1.0.1-beta installs from source. Production deploy uses install-all.sh; development uses nexus.sh.

Linux x86_64

git clone https://github.com/ZacharyGeurts/AmmoOS.git
cd AmmoOS
sudo ./install-all.sh

Browser opens http://127.0.0.1:9477/field on start.

Development tree

export SG_ROOT=/path/to/SG
./scripts/ammoos-beta-pipeline.sh
./nexus.sh

Windows

# PowerShell (see stealth.ps1 in release assets)
# Or WSL2:
tar -xzf ammooos-1.0.1-beta-source.tar.gz && cd ammooos-1.0.1-beta
sudo ./install-all.sh

Verify

./scripts/ammoos-launch-verify.sh
