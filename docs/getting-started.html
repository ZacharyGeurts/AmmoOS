Getting Started — Installers

One admin approval installs the full field stack: daemon, panel, firewall, polkit, Tristate / Underlay F9, and desktop entries. Browser opens automatically on first boot.

Download release

From v10.4.0 release:

AssetUse
nexus-shield-10.4.0-source.tar.gzFull tree — recommended for install
nexus-shield-10.4.0-installers.tar.gzScripts only — overlay into existing tree

Full install (Linux)

tar -xzf nexus-shield-10.4.0-source.tar.gz
cd nexus-shield-10.4.0
chmod +x install-all.sh genius_shield.sh nexus.sh nexus-install-gui.sh
sudo ./install-all.sh

This runs genius_shield.sh which:

  • Copies to /usr/local/lib/nexus-shield
  • Enables nexus-genius.service (systemd)
  • Runs first-install boot-impl
  • Installs desktop entries + Tristate installer shortcut

Tristate / Underlay F9

Permanent underlay installer — F9 hotkey in panel, no off switch after commit. GUI at:

http://127.0.0.1:9477/underlay-f9?sector=underlay

Legacy path /tristate-installer redirects to the same surface.

Verify

nexus status
nexus verify
curl -s http://127.0.0.1:9477/api/status | jq .
